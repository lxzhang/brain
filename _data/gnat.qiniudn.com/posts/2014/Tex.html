guide star(int n = 5, real r0 = 1)
{
    guide unitstar;
    if (n < 5) return nullpath;
    real theta = 180/n;
    real r = Cos(2theta) / Cos(theta);
    for (int k = 0; k < n; ++k)
        unitstar = unitstar -- dir(90+2k*theta)
            -- r * dir(90+(2k+1)*theta);
    unitstar = unitstar -- cycle;
    return scale(r0) * unitstar;
}

    \End

C 语言的循环语句
================

C 语言可以使用 `for` 循环语句，例如计算 $s = \sum_k k^2$：

    int s = 0;
    for (int k = 1; k < 10; ++k)
        s += k * k;
